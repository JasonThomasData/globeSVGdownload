Country shape SVG exporter

A tool for artists and non-programmers to export SVG shapes of countries

This globe is a better representation of countries' shapes than projections that depict the world as flat. These shapes are useful for seeing countries in isolation or comparing land masses. These shapes will not snap together. For snapping shapes, there are many world projections like Mercator and Robinson.

I'm not anti-Mercator, but this website shows it has limitations. Here's some reading material on projections - XKCD, Britanica.

The original code is a bl.ocks project by KoGor.
